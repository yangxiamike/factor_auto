# Program

1. Read the experiment spec, memory, and research notes.
2. Append 30 new candidates to `candidate_factors/candidates.jsonl`.
3. Run `fm factor validate`.
4. Run `fm factor evaluate`.
5. Read `runs/{run_id}/summary.md` and `runs/{run_id}/logs/evaluate.log`.
6. Update `codex/research_notes.md`.
7. Update `codex/memory.md` only after repeated stable insight.
